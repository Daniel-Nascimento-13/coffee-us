# Projeto "Coffee Us"

Este repositório reúne *uma landing page*, voltada para a divulgação da *Coffee Us*, uma empresa especializada em *serviços de café para eventos*, consolidada há mais de *5 anos no mercado*.

O projeto foi criado com o objetivo de *aumentar a conversão*, transformar acessos em *oportunidades reais* e potencializar o *fechamento de contratos*.

A landing page foi desenvolvida para conectar o cliente diretamente ao atendimento, oferecendo uma experiência *prática, rápida e envolvente*, além de apresentar os serviços de forma estratégica, com foco em *gerar desejo*, *facilitar a tomada de decisão* e conduzir o usuário até o contato com *informações pré-estruturadas*, sendo um projeto *ativo e em uso real*, e não apenas uma simulação.

O site é *destinado* a empresas, organizadores de eventos e marcas que buscam oferecer uma *experiência diferenciada com café*, permitindo conhecer os serviços, entender os diferenciais e realizar o contato de forma simples, sendo direcionados para o *WhatsApp*, onde o atendimento é finalizado de forma *personalizada e estratégica*.
